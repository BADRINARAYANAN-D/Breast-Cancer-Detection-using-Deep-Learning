# Breast Cancer Detection using Deep Learning (Medical Image Classification)

## Overview
Early detection of breast cancer plays a vital role in improving patient survival rates.  
This project explores the progression from machine learning to deep learning for breast cancer classification, demonstrating how deep learning models capture complex patterns more effectively.

## Problem Statement
To classify breast cancer cases as:
- Benign
- Malignant

using diagnostic features and deep learning techniques.

## Learning Progression
- Started with traditional machine learning understanding
- Applied feature standardization
- Transitioned to Deep Learning using Neural Networks
- Compared learning behavior and performance

## Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- TensorFlow / Keras

## Dataset
- Breast Cancer Dataset (Scikit-learn)
- 30 diagnostic features

## Model Architecture
- Input Flatten Layer
- Dense Hidden Layer with ReLU activation
- Output Layer with Sigmoid activation

## Model Evaluation
- Training and validation accuracy and loss visualization
- Test accuracy evaluation
- Prediction on custom input data

## Why Computer Vision
This project focuses on medical image classification, a core application of Computer Vision, using Deep Learning techniques to assist healthcare diagnostics.

## Key Learnings
- Difference between machine learning and deep learning approaches
- Overfitting vs generalization
- Importance of feature scaling in deep learning models

## Future Scope
- Extend to CNN-based medical image datasets
- Integrate with real-time clinical decision systems
- Improve robustness using advanced architectures
